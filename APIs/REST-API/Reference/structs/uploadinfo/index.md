# UploadInfo

Contains the information on a temporary upload. 

## Properties

{:.struct.table}
Type | Name | Description
--- | --- | ---
`int` | UploadID | Temporary upload ID given by the server. 
`string` | Title | File name without extension. 
`string` | Extension | File extension. 
`long` | Size | File size.
