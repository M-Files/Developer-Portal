VaultValueListItemOperations Object Members
Overview 
Public Methods
 AddValueListItemAdds a new value list item to a value list.
 ChangeAutomaticPermissionsToACLChanges the automatic permissions for the specified value list item. The specified automatic permissions will be applied to an object when the user selects this value list item in the metadata card of the object.
 ChangeAutomaticPermissionsToACLExChanges the automatic permissions for the specified value list item.
 ChangeAutomaticPermissionsToItemsOwnPermissionsChanges the automatic permissions for the specified value list item to match those of its own permissions. The specified automatic permissions will be applied to an object when the user selects this value list item in the metadata card of the object.
 ChangeAutomaticPermissionsToItemsOwnPermissionsExChanges the automatic permissions for the specified value list item to match those of its own permissions.
 ChangeAutomaticPermissionsToNamedACLChanges the automatic permissions for the specified value list item. The specified automatic permissions will be applied to an object when the user selects this value list item in the metadata card of the object.
 ChangeAutomaticPermissionsToNamedACLExChanges the automatic permissions for the specified value list item.
 ChangePermissionsToACLChanges the permissions for the specified value list item.
 ChangePermissionsToACLExChanges the permissions for the specified value list item.
 ChangePermissionsToNamedACLChanges the permissions for the specified value list item.
 ChangePermissionsToNamedACLExChanges the permissions for the specified value list item.
 ClearAutomaticPermissionsClears the automatic permissions from the specified value list item. This prevents any permissions from being set to an object as a result of this value list item being selected.
 ClearAutomaticPermissionsExClears the automatic permissions of the specified value list item.
 GetReferencedValueListItemsInIdOrderGets value list items that are referenced by one or more documents (possibly limiting the set of documents with conditions) in ID order.
 GetValueListItemByDisplayIDGets a value list item from the specified value list, using the display ID (external ID) of the item.
 GetValueListItemByDisplayIDExGets a value list item from the specified value list, using the display ID (external ID) of the item.
 GetValueListItemByGUIDGets a value list item from the specified value list by using the GUID of the item.
 GetValueListItemByIDGets a value list item from the specified value list, using the internal ID of the item.
 GetValueListItemByIDExGets a value list item from the specified value list, using the internal ID of the item.
 GetValueListItemByIDEx2Gets a value list item from the specified value list, using the internal ID of the item.
 GetValueListItemIDByGUIDGets the ID of the value list item with the specified GUID. Returns -1 if a match is not found.
 GetValueListItemsGets the value list items of a value list.
 GetValueListItemsExGets the value list items of a value list.
 GetValueListItemsEx2Gets the value list items of a value list.
 GetValueListItemsInIdOrderGets the items of a value list ordered by ID.
 GetValueListItemsWithPermissionsGets the value list items of a value list.
 RemoveValueListItemDeletes a value list item.
 RemoveValueListItemExDeletes a value list item.
 SearchForValueListItemsExSearches for value list items in the specified value list
 SearchForValueListItemsEx2Searches for value list items in the specified value list
 SearchForValueListItemsWithPermissionsSearches for value list items in the specified value list
 UndeleteValueListItemUndeletes a previously deleted value list item.
 UndeleteValueListItemExUndeletes a previously deleted value list item.
 UpdateValueListItemUpdates the value list item.
