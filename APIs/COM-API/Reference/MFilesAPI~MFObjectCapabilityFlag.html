MFObjectCapabilityFlag Enumeration
Description
Bit flags that describe the capabilities of an object.
Members
MemberValueDescription
MFObjectCapabilityFlagAddAndRemoveFiles8192Adding and removing files is allowed.
MFObjectCapabilityFlagAllCapabilities-1All object capabilities.
MFObjectCapabilityFlagCanBeDeleted8The object can be deleted.
MFObjectCapabilityFlagCanBeDestroyed16The object can be destroyed.
MFObjectCapabilityFlagCanBeRenamed64The object can be renamed
MFObjectCapabilityFlagCanBeUndeleted32The object can be undeleted.
MFObjectCapabilityFlagCanChangeFileType16384Changing the type of the files contained in the object is allowed. Requires M-Files 19.2.7378.0 or later.
MFObjectCapabilityFlagCanHaveHistory1The object can have history.
MFObjectCapabilityFlagCanHaveRelationships2The object can have relationships.
MFObjectCapabilityFlagEditableAssignments256The assignment functions are available for the object.
MFObjectCapabilityFlagEditableFiles1024The files of the object can be edited.
MFObjectCapabilityFlagEditableMetadata128The metadata of the object can be edited.
MFObjectCapabilityFlagEditableOldVersionDetails4The old version details of the object are editable.
MFObjectCapabilityFlagEditableVersionComments2048The version comments of the object can be edited.
MFObjectCapabilityFlagEditableWorkflows512The workflow capabilities are available for the object.
MFObjectCapabilityFlagNone0No object capabilities.
MFObjectCapabilityFlagSingleFileMultiFileConversion4096The object can be converted from a single-file document to a multi-file document and conversely from a multi-file document to a single-file document.
