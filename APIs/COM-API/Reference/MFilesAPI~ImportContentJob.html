ImportContentJob Object
Members 
Description
The ImportContentJob class represents a content import job.
Public Methods
 CloneCreates a clone of this object.
 DisableFeaturesRequiringSystemAdministratorRoleDisables all the features that require the system administrator role for importing. The number of features that are disabled may increase in the future.
 SetReplicationViaCloudSets the information for replication via cloud storage.
Public Properties
 ActivateAutomaticPermissionsForNewOrChangedDefinitionsSpecifies whether to activate automatic permissions of objects due to new or changed definitions that affect them.
 CombinedConnectionStringThe connection string combined from external storage location and target folder name.
 ConfigurationIDThe configuration ID of this import job.
 ConnectionStringThe external storage location access information.
 DisableCustomCodeInImportedPropertyDefinitionsSpecifies whether all the custom code in imported property definitions is deactivated.
 DisableCustomCodeInImportedWorkflowsSpecifies whether all the custom code in imported workflows is deactivated.
 DisableImportedExternalObjectTypeConnectionsSpecifies whether all the external connections of the imported object types are disabled.
 DisableImportedExternalUserGroupsSpecifies whether all the external connections of the imported user groups are disabled.
 DisableImportedVaultEventHandlersSpecifies whether the all imported event handlers are deactivated.
 ExternalFolderNameThe target folder in the external cloud storage.
 FlagsSpecifies additional options as bit flags.
 IgnoreAutomaticPermissionsDefinedByObjectsSpecifies whether to ignore the automatic permissions defined by the imported objects.
 ImpersonationImpersonation information to specify the access to the target files.
 MetadataStructureSelectorsSpecifies the metadata structure items that should be imported.
 PermissionsSpecifies the access control list set by this job.
 SourceLocationThe full path to the source location on server.
 UseDefaultObjectTypePermissionsSpecifies whether to use the default permissions of object types. The UsePermissions property has precedence over this property.
 UseNamesAsAliasesSpecifies whether the incoming name of a structure item is used as its alias if no other alias is available.
 UsePermissionsSpecifies whether to use the Permissions value.
Remarks

 

A content import job as seen in M-Files Admin.
