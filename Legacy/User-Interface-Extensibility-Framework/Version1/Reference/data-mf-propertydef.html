M-Files UI Extensibility Framework
data-mf-propertydef Attribute
 Description
Specifies the metadata binding for the control.
 Remarks

The value of the appribute binds the control to the property value of an object.

  • Example
<span class="mf-control mf-text" data-mf-propertydef="1042">
</span>
 Applies for
<div> Element with mf-control Class.
