OnNewCommonDialogShellFrame Event
M-Files UI Extensibility Framework > Events > OnNewCommonDialogShellFrame Event
 Description
Triggered when a shell frame object is created in a common dialog.
 Parameters

The new shell frame object.

 Return Type

Object [ optional ]

Event handler collection object for the new object.

 Remarks
