LegacyConfigurableVaultApplicationBaseTSecureConfiguration Class

Legacy implementation of the common base class for applications that want to use a single, securable configuration with full MFAdmin support. - Uses the old TaskQueueManager from MultiserverMode namespace. - Uses old OnConfigurationUpdated signature.
