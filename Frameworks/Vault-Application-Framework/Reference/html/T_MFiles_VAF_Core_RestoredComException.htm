RestoredComException Class

Used to rethrow COM exceptions emitted from MFServer call that replaced/obscured a .NET exception, while preserving and exposing the original .NET exception.

Remarks
This exception's message matches the COM exception's (as that includes the stack trace, and MFServer may add to it as it bubbles up the stack). The COM exception is set as the InnerException, and the original .NET exception is available on the OriginalException property.
