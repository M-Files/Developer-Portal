DynamicListBaseAttributeRootAncestorOffset Property

Used to calculate the root ancestor path to which this reference-able item has been defined for. Example: ThisItem( Offset = 0) -> ParentItem( Offset 1 ) -> GrandParentItem( Offset = 2 ) When no offset is defined, the current item will be used as the root item. Note: This is only used when the subscriber is filtered.
