M-Files UI Extensibility Framework
OnSearchCriteriaChanged Event
M-Files UI Extensibility Framework > Events > OnSearchCriteriaChanged Event
 Description
This event is triggered when the search criteria has changed.
 Parameters
forceRefresh : Boolean [ in ]

If set to true, the user interface processes the event and assumes responsibility for calling the UpdateSearch method to update the search results.

 Remarks
Supported in M-Files 2018 and later.
