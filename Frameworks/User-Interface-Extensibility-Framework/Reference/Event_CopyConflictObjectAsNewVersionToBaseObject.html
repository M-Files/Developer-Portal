M-Files UI Extensibility Framework
OnCopyConflictObjectAsNewVersionToBaseObject Event
M-Files UI Extensibility Framework > Events > OnCopyConflictObjectAsNewVersionToBaseObject Event
 Description
Triggered when M-Files starts to create a copy of the conflict object as a new version of the related base object.
 Parameters

BaseObjVer : IObjVer [ in ]

The latest version of the base object for which the new version is created.

ConflictObjVer : IObjVer [ in ]

The latest version of the related conflict object that is copied. The object can be in the deleted state.

 Remarks

Supported in M-Files 21.3.10021.4 and later.
