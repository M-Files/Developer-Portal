M-Files UI Extensibility Framework
OnDragEntered Event
M-Files UI Extensibility Framework > Events > OnDragEntered Event
 Description
This event is triggered when the dashboard has its DropTarget property set to "true" and the user drags objects inside the dashboard.
 Parameters
items : IShellItems Object [ in ]

Contains the items that are being dragged.

 Remarks
Supported in M-Files 2018 and later.
