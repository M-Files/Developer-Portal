M-Files UI Extensibility Framework
OnObjectCheckoutUndone Event
M-Files UI Extensibility Framework > Events > OnObjectCheckoutUndone Event
 Description
Triggered after a request to undo checkout of a vault object is succeeded on M-Files Server.
https://developer.m-files.com/APIs/COM-API/Reference/
 Parameters

objectVersion : IObjectVersion [ in ]

The restored last visible object version.
 Remarks

The OnObjectCheckoutsUndone Event is triggered, too.

Supported in M-Files 9.0.3372.6 and newer.
