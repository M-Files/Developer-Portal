M-Files UI Extensibility Framework
mf-nodropdown Class
 Description
Prevents the drop-down menu.
 Remarks
The drop down menu is not shown for the control. Value can be edited by entering text.
 Applies for
<span> Element with mf-lookup Class or mf-multiselectlookup Class.
