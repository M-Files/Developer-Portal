M-Files UI Extensibility Framework
mf-objectversion Class
 Description
A specifies that the element is a help text container for metadata control.
 Remarks

The element content specifies the help text. Must be placed inside an element with mf-control class.

  • Example
<span class="mf-control mf-lookup" data-mf-propertydef="100">
<span class="mf-helptext" style="color:#999999">[Class]</span>
</span>
 Applies for
<span> Element
