M-Files UI Extensibility Framework
OnObjectFileAdded Event
M-Files UI Extensibility Framework > Events > OnObjectFileAdded Event
 Description
Triggered after a request to add a new object file is succeeded on M-Files Server.
https://developer.m-files.com/APIs/COM-API/Reference/
 Parameters

objectVersion : IObjectVersion [ in ]

The object version updated in the operation. This object version contains the added file.

objectFile : IObjectFile [ in ]

Identifies the new object file created in the operation.

 Remarks

Supported in M-Files 9.0.3372.6 and newer.
